class SocketAddress

package js.node.net

@:jsRequire("net","SocketAddress")

Represents a network endpoint as an IP address and port pair.

See also:

  • https://nodejs.org/api/net.html#class-netsocketaddress

Static methods

staticparse(input:String):Null<SocketAddress>

Parses an IP address and optional port string into a SocketAddress.

input examples: 123.1.2.3:1234 or [1::1]:1234. Returns null/undefined if parsing fails.

See also:

  • https://nodejs.org/api/net.html#socketaddressparseinput

Constructor

new(?options:{port:Null<Int>, flowlabel:Null<Int>, family:Null<String>, address:Null<String>})

Creates a new SocketAddress.

family is either 'ipv4' or 'ipv6'. Default: 'ipv4'.

See also:

  • https://nodejs.org/api/net.html#new-netsocketaddressoptions

Variables

read onlyaddress:String

The network address as either an IPv4 or IPv6 string.

read onlyfamily:String

Either 'ipv4' or 'ipv6'.

read onlyflowlabel:Int

An IPv6 flow-label used only if family is 'ipv6'.

read onlyport:Int

An IP port.
