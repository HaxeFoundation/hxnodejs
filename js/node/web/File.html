class File

package js.node.web

extends Blob

@:native("File")

Provides information about files. Extends Blob.

Available as a global and via require('node:buffer').File.

See also:

  • https://nodejs.org/api/buffer.html#class-file

  • https://nodejs.org/api/globals.html#class-file
