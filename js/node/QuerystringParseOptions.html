typedef QuerystringParseOptions

package js.node

import js.node.Querystring

Options used for Querystring.parse method.

Properties

maxKeys:Null<Int>

Specifies the maximum number of keys to parse. Defaults to 1000. Specify 0 to remove key counting limitations.

decodeURIComponent:Null<String ‑> String>

The function to use when decoding percent-encoded characters in the query string. Defaults to Querystring.unescape.
