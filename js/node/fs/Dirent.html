class Dirent

package js.node.fs

A representation of a directory entry, which can be a file or a subdirectory within the directory, as returned by reading from an Dir or from Fs.readdir / Fs.glob when withFileTypes is true.

Variables

name:EitherType<String, Buffer>

The file name that this Dirent object refers to. The type of this value is determined by the encoding option.

parentPath:String

The path to the parent directory of the file this Dirent object refers to.

Methods

isBlockDevice():Bool

Returns true if this Dirent describes a block device.

isCharacterDevice():Bool

Returns true if this Dirent describes a character device.

isDirectory():Bool

Returns true if this Dirent describes a file system directory.

isFIFO():Bool

Returns true if this Dirent describes a first-in-first-out (FIFO) pipe.

isFile():Bool

Returns true if this Dirent describes a regular file.

isSocket():Bool

Returns true if this Dirent describes a socket.

isSymbolicLink():Bool

Returns true if this Dirent describes a symbolic link.
